epimaker

どんなサイト？

  • LINEやGoogleを使って会員登録できるwebアプリ。
  • エピソードをAIを使って作成できる。
  • エピソードを公開して、いいねしたりコメントしたりできる。
  • いいねランキングで上位を狙う。

技術仕様

  1. Laravel10を使用しています。
  2. LINEやgoogleでの認証にOAuthを使用しています。
  3. Twitterでも認証でき、作成したエピソードをTweetできます。
  4. javascriptを使用してOpenAIへリクエストを送り、AIでエピソードを自動作成します。
  5. ユーザーロールをadminとuserに分け、adminユーザーだけが管理タスクを実行できます。
  6. viewはbladeを使用しています。
