epimaker

どんなサイト？

  • LINEやGoogleを使って会員登録できるwebアプリ。
  • エピソードをAIを使って作成できる。
  • エピソードを公開して、いいねしたりコメントしたりできる。
  • いいねランキングで上位を狙う。

技術仕様

  1. Laravel10を使用しています。
  2. LINEやgoogleでの認証にOAuthを使用しています。
  3. OpenAIへjavascriptを使用してリクエストを送ります。
  4. viewはbladeを使用しています。
